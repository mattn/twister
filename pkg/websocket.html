func (*Conn) ReadMessage

func (conn *Conn) ReadMessage() (chunk []byte, hasMore bool, err os.Error)

ReadMessage reads a message from the client. The message is returned in one or more chunks. hasMore is set to false on the last chunk of the message. If the message fits in the read buffer size specified in the call to Upgrade, then the message is guaranteed to be returned in a single chunk. The returned chunk points to the internal state of the connection and is only valid until the next call to ReadMessage.

func (*Conn) WriteMessage

func (conn *Conn) WriteMessage(p []byte) os.Error

WriteMessage write a message to the client. The message cannot contain the bytes with value 0 or 127.
