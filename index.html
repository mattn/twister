Twister

Twister is an HTTP server and framework for the Go programming language.

Installation

  1. Install Go.
  2. goinstall github.com/garyburd/twister/server

The Go distribution is Twister's only dependency.

Packages

  • web - Defines the application interface to a server and includes functionality used by most web applications.
    • Routing to handlers using regular expression match on host and path.
    • Protection against cross site request forgery.
    • Extensible model for middleware.
    • Cookie parsing.
    • WebSockets.
    • Static file handling.
    • Signed values for cookies and form parameters.
    • Multipart forms.
  • server - An HTTP server impelemented in Go.
  • oauth - OAuth client.
  • expvar - Exports variables as JSON over HTTP for monitoring.
  • websocket - WebSocket server implementation.

Documentation

API Reference: web, server, oauth, expvar, websocket.

Examples

  • examples/wiki - The Go web application example converted to use Twister instead of the Go http package.
  • examples/demo - Illustrates basic features of Twister.
  • examples/twitter - Login to Twitter with OAuth and display home timeline.
  • examples/facebook - Login to Facebook with OAuth2 and display news feed.
